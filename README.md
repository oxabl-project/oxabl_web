# oxabl-web

Marketing site for [Oxabl](https://github.com/oxabl-project/oxabl) — high-performance tooling for Progress ABL, written in Rust.

Part of the [`<kbd>` tag](https://github.com/evanbrobertson) brand family.

## Stack

- [Astro](https://astro.build) 6
- React islands (HoverGlow)
- Tailwind CSS 4 + shadcn/ui (base-luma)
- Geist / IBM Plex Sans / JetBrains Mono

## Develop

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
pnpm preview
```

## Brand notes

- Logo wordmark is `ox_abl` — green underscore is intentional
- Palette is kbd-tag neutrals + an ABL-adjacent green accent (not Progress brand colors)
- Keyboard shortcuts (`h` / `f` / `p` / `s`) match the kbd-tag nav language
- Footer states the project is unofficial and unaffiliated
